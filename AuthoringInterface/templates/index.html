AniCode: Authoring Coded Artifacts for Network-Free Personalized Animations

Zeyu Wang*, Shiyu Qiu*, Qingyang Chen, Natallia Trayan, Alexander Ringlein, Julie Dorsey, Holly Rushmeier

Yale University

Published in the Visual Computer and presented at Computer Graphics International 2019

To use the online demo of AniCode, please follow the steps below:

  1. Print a reference QR code and put it on an object or in a scene with which you would like to author an animation.
  2. Take a picture of the object or scene and upload it to our authoring interface. Apply animation schemes and generate an authored QR code.
  3. Print the authored QR code and overlay it on the reference QR code. Use our client-side mobile application (Android) to view the authored animation.

Please refer to our paper, slides, and source code for more details.
